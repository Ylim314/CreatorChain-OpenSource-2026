# AI模型配置更新说明 (2025年12月)

## 📌 更新概览

本次更新将CreatorChain项目的AI模型配置升级到2025年最新版本,修复了模型版本过时、API地址错误、功能配置错误等关键问题。

---

## 🔥 主要更新内容

### 1. 通义千问 (Qwen) - 阿里云

#### ✅ 更新前的问题
- ❌ 使用过时模型: `qwen-turbo`, `qwen-plus`, `qwen-max`, `qwen-vl-plus`
- ❌ 图像生成配置错误: `qwen-vl-plus`是视觉理解模型,不是图像生成模型
- ❌ API地址不完整

#### ✅ 更新后的配置

**文本生成模型 (2025最新)**
- `qwen3-max` - 最强旗舰模型 (2025.09发布)
  - 上下文: 262,144 tokens (26万)
  - 能力: 复杂任务、推理、代码、多语言
  - 定价: 0.0032元/千tokens (输入)
  
- `qwen-plus` - 均衡版 (推荐)
  - 上下文: 1,000,000 tokens (100万超长上下文!)
  - 能力: 成本/性能均衡
  - 定价: 0.0008元/千tokens (输入)
  
- `qwen-flash` - 极速版 (推荐用于简单任务)
  - 上下文: 1,000,000 tokens
  - 能力: 速度快、成本极低
  - 定价: 0.00015元/千tokens (输入)
  - 注意: 建议用此模型替代已废弃的`qwen-turbo`

**视觉理解模型 (图片转文字)**
- `qwen3-vl-plus` - 旗舰视觉模型
- `qwen3-vl-flash` - 极速视觉模型

**图像生成模型 (文字转图片)** ⭐ 答辩推荐使用
- `qwen-image-plus` - 通义千问文生图Plus版
  - 特色: 中英文文本渲染能力突出
  - 定价: 0.2元/张
  - 适用: 需要图片中包含文字的场景
  
- `wan2.5-t2i-preview` - 通义万相2.5预览版
  - 特色: 艺术创作、风格丰富、影视级画质
  - 定价: 480P: 0.3元/秒, 720P: 0.6元/秒, 1080P: 1元/秒
  - 适用: 高质量艺术创作

**API地址更新**
```
基础URL: https://dashscope.aliyuncs.com/api/v1
文本生成: /services/aigc/text-generation/generation
图像生成: /services/aigc/text2image/image-synthesis
```

**官方文档**
https://help.aliyun.com/zh/model-studio/getting-started/models

### 2. 豆包 (Doubao) - 字节跳动 🆕

#### ✅ 新增完整配置

项目之前**完全缺失**豆包模型配置,现已完整添加:

**文本生成模型**
- `doubao-pro-32k` - 专业版
  - 上下文: 32,000 tokens
  - 特点: 性能强劲、适合复杂任务
  
- `doubao-lite-32k` - 轻量版 (推荐)
  - 上下文: 32,000 tokens
  - 特点: 经济实惠、性价比高
  
- `doubao-character-32k` - 角色扮演版
  - 上下文: 32,000 tokens
  - 特点: 擅长角色扮演、对话生成

**视觉理解模型**
- `doubao-vision-pro` - 图文理解专业版
  - 能力: 图片识别、OCR、图文问答

**API地址**
```
基础URL: https://ark.cn-beijing.volces.com/api/v3
```

**官方文档**
https://www.volcengine.com/docs/82379/1302050

**注意事项**
- 豆包需要在火山引擎控制台创建推理接入点
- 每个模型需要单独配置endpoint_id
- API认证使用Bearer Token方式

## ❓ 常见问题

**Q: 为什么不用ChatGPT?**
A: 1) 国产模型更符合合规要求; 2) 通义千问和豆包性价比更高; 3) 答辩时展示国产技术更有优势

**Q: 图像生成哪个模型最好?**
A: 看需求。需要文字渲染用`qwen-image-plus`,纯艺术创作用`wan2.5-t2i-preview`

**Q: 豆包需要什么额外配置?**
A: 需要在火山引擎控制台创建推理接入点,获取endpoint_id和API Key

**Q: 旧的模型配置还能用吗?**
A: 可以,我们保留了向后兼容,但建议尽快迁移到新版本
