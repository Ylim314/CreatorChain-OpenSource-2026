# 作品简介

CreatorChain 是一个面向数字创作场景的确权平台，目标是将创作过程、最终作品、链上登记与分布式存储打通，形成可验证、可追溯的版权证明链路。

## 核心能力

- 多元创作入口：支持 AI 生成、人工创作、混合作品。
- 双重确权流程：记录创作过程并确认最终版本，减少归属争议。
- 链上与链下协同：合约负责确权与授权，后端负责业务编排与风控。
- IPFS 存证：关键内容哈希与元数据可追溯。
- 积分机制：用于平台内激励与功能消费，不作为加密货币发行。

## 技术组成

- 前端：React + Ethers.js
- 后端：Go + Gin + GORM
- 合约：Solidity + Hardhat
- 存储：IPFS + 关系型数据库

## 适用场景

- 个人创作者发布与版权存证
- 团队协作内容的贡献追踪
- 教学/比赛中的区块链确权演示

## 当前状态

本仓库为比赛项目开源版本，优先保证可阅读、可运行、可复现。部分模块仍在持续整理和演进中。
