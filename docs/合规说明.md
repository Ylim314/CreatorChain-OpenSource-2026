# CreatorChain 合规说明

## 重要声明

本项目严格遵守相关法律法规，不涉及任何虚拟货币或代币交易。

## 合规设计

### 1. 积分系统（合规）

- **性质**：平台内激励机制，类似游戏积分、会员积分
- **存储**：存储在PostgreSQL/SQLite数据库中，不涉及区块链
- **获取方式**：
  - 系统奖励（创作作品、完成确权、用户互动等）
  - 用户间转账（平台内）
- **使用范围**：
  - 购买作品授权
  - 平台内功能解锁
- **限制**：
  - 积分不能兑换成法币
  - 积分不能兑换成任何形式的虚拟货币
  - 积分不能提现
  - 积分仅限平台内使用

### 2. NFT确权（合规）

- **性质**：仅作为版权证明，不涉及交易
- **功能**：记录作品版权信息，确权存证
- **存储**：版权信息存储在区块链上，永久保存
- **限制**：
  - NFT不进行交易
  - NFT不涉及任何货币价值
  - NFT仅作为版权证书使用

### 3. 智能合约（合规）

- **实际部署**：
  - CreatorNFT.sol - 版权确权NFT（仅作为版权证明）
- **设计原型（不实际部署）**：
  - CreatorToken.sol - 设计原型，不部署
  - LicenseManager.sol - 设计原型，不部署
  - CreatorDAO.sol - 设计原型，不部署
- **说明**：设计原型合约仅作为技术参考，不实际部署到任何区块链网络

### 4. 实际实现

- **授权购买**：通过后端API实现，使用链下积分系统
- **积分管理**：通过后端API实现，存储在数据库中
- **治理功能**：通过后端API实现，使用链下积分系统
- **版权确权**：通过CreatorNFT合约实现，仅作为版权证明

## 合规要点

1. ✅ **不涉及虚拟货币**：项目不使用任何形式的虚拟货币或代币
2. ✅ **积分系统合规**：积分仅用于平台内激励，不能兑换法币
3. ✅ **NFT仅作证明**：NFT仅作为版权证明，不进行交易
4. ✅ **智能合约合规**：仅部署版权确权合约，不部署代币相关合约
5. ✅ **实际功能合规**：所有涉及支付的功能均使用链下积分系统
