# CreatorChain 作品安装说明

## 系统要求

### 硬件要求

- **CPU**: 4 核心以上 (推荐 8 核心)
- **内存**: 8GB 以上 (推荐 16GB)
- **存储**: 20GB 可用空间 (推荐 50GB)
- **网络**: 稳定的互联网连接 (推荐 100Mbps+)

### 软件要求

- **操作系统**: Windows 10/11, macOS 10.15+, Ubuntu 20.04+
- **Node.js**: 18.0+ (推荐 18.x LTS)
- **Go**: 1.24+ (推荐 1.24.7)
- **Git**: 2.0+
- **数据库**: MySQL 8.0+ 或 SQLite 3.0+ (默认)
- **Redis**: 6+ (可选，支持内存缓存)
- **IPFS**: 本地节点或 Pinata API

## 快速安装

### 1. 克隆项目

目前仅支持通过网盘链接下载

```bash
# 下载项目压缩包并解压
# 进入项目目录
cd CreatorChain
```

### 2. 安装依赖

#### 安装前端依赖

```bash
cd client
npm install
cd ..
```

#### 安装后端依赖

```bash
cd backend
go mod tidy
go mod download
cd ..
```

#### 安装智能合约依赖

```bash
cd contracts
npm install
cd ..
```

### 3. 环境配置

#### 创建环境变量文件

```bash
# 在backend目录下创建.env文件
cd backend
cp .env.example .env
```

#### 配置环境变量

```bash
# .env文件内容
# 数据库配置 (SQLite - 默认配置)
DATABASE_URL=sqlite:///./creatorchain.db

# 或使用 MySQL
# DATABASE_URL=mysql://root:720720@localhost:3306/creatorchain?charset=utf8mb4&parseTime=True&loc=Local

# Redis配置 (可选，支持内存缓存)
REDIS_URL=redis://localhost:6379

# 区块链配置
ETHEREUM_RPC=https://eth-mainnet.alchemyapi.io/v2/your-key
POLYGON_RPC_URL=https://polygon-rpc.com
PRIVATE_KEY=your_private_key_here

# AI引擎配置
AI_API_KEY=your_ai_api_key
AI_BASE_URL=https://api.openai.com

# IPFS配置
IPFS_GATEWAY=https://ipfs.io/ipfs/
PINATA_API_KEY=your_pinata_api_key
PINATA_SECRET_KEY=your_pinata_secret_key

# 服务配置
PORT=8080
```

### 4. 数据库初始化

#### 使用 SQLite (默认，无需额外配置)

```bash
# SQLite会自动创建数据库文件
```

#### 使用 MySQL (可选)

```bash
# 创建数据库
mysql -u root -p -e "CREATE DATABASE creatorchain CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;"

# 程序启动时会自动运行数据库迁移
```

### 5. 启动服务

#### 方法一：使用批处理脚本（推荐）

**一键启动整个项目**：

```bash
# 在项目根目录下运行 - 一键启动
start.bat
```


#### 方法二：手动启动

**启动后端服务**：

```bash
cd backend
go mod tidy
go run cmd/api/main.go
```

**启动前端服务**（新终端窗口）：

```bash
cd client
npm start
```

#### 编译智能合约 (可选)

```bash
cd contracts
npx hardhat compile
```

## 详细安装步骤

### 前端安装详解

#### 1. Node.js 环境准备

```bash
# 检查Node.js版本
node --version
npm --version

# 如果版本过低，请升级到最新LTS版本
# Windows: 从官网下载安装包
# macOS: brew install node
# Ubuntu: curl -fsSL https://deb.nodesource.com/setup_18.x | sudo -E bash -
```

#### 2. 前端依赖安装

```bash
cd client

# 安装生产依赖
npm install

# 安装开发依赖 (如果需要)
npm install --save-dev
```

#### 3. 前端配置

```bash
# 创建前端环境变量文件
cp .env.example .env.local

# 配置API地址
echo "REACT_APP_API_URL=http://localhost:8080" >> .env.local
echo "REACT_APP_CHAIN_ID=5" >> .env.local  # Goerli测试网
```

### 后端安装详解

#### 1. Go 环境准备

```bash
# 检查Go版本
go version

# 如果版本过低，请升级到1.24+
# Windows: 从官网下载安装包
# macOS: brew install go
# Ubuntu: sudo apt install golang-go
```

#### 2. 后端依赖安装

```bash
cd backend

# 下载依赖
go mod tidy
go mod download

# 验证依赖
go mod verify
```

#### 3. 数据库配置

##### SQLite 配置 (推荐新手)

```bash
# 无需额外配置，会自动创建数据库文件
# 数据库文件位置: backend/data/creatorchain.db
```

##### MySQL 配置 (推荐生产环境)

```bash
# 安装MySQL
# Windows: 从官网下载安装包
# macOS: brew install mysql
# Ubuntu: sudo apt install mysql-server

# 启动MySQL服务
sudo systemctl start mysql

# 创建数据库
mysql -u root -p -e "CREATE DATABASE creatorchain CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;"

# 配置数据库连接
# 编辑backend/.env文件
DATABASE_URL=mysql://root:password@localhost:3306/creatorchain?charset=utf8mb4&parseTime=True&loc=Local
```

## 常见问题解决

### 1. 端口冲突

```bash
# 检查端口占用
netstat -ano | findstr :3000
netstat -ano | findstr :8080

# 杀死占用进程
taskkill /PID <进程ID> /F
```

### 2. 依赖安装失败

```bash
# 清理缓存
npm cache clean --force
go clean -modcache

# 重新安装
rm -rf node_modules package-lock.json
npm install
```

### 3. 数据库连接失败

```bash
# 检查数据库服务状态
# MySQL
sudo systemctl status mysql

# SQLite (无需服务，检查文件权限)
ls -la backend/creatorchain.db

# 检查连接参数
# 确认.env文件中的数据库URL正确
```

### 4. 区块链连接失败

```bash
# 检查网络连接
ping polygon-rpc.com

# 检查API密钥
# 确认.env文件中的POLYGON_RPC_URL有效
```

### 2. 功能测试

#### 基础功能测试

1. **用户注册/登录**
   - 访问前端页面 http://localhost:3000
   - 连接MetaMask钱包
   - 完成用户注册
   - 验证积分奖励 (1000积分)

2. **双重确权流程测试**
   - **第一次确权**: 记录创作过程
     - 上传作品文件（AI生成或人工创作）
     - 填写创作信息（AI模型信息或创作工具）
     - 记录创作过程（提示词和参数或创作步骤）
     - 生成Merkle树根
   - **第二次确权**: 确认最终作品
     - 确认作品内容
     - 生成验证证明
     - 记录到区块链
     - 获得版权证书

3. **多元化创作功能**
   - **AI生成创作**: 使用AI模型生成内容，智能分析贡献度
   - **手工创作**: 上传传统手工制作的作品，记录创作过程
   - **混合创作**: AI辅助的人工创作，记录双重创作贡献
   - **内容存储**: 自动上传到IPFS，获得永久分布式存储

4. **市场功能**
   - 浏览作品列表
   - 购买作品 (使用积分)
   - 查看交易记录
   - 管理收藏夹
