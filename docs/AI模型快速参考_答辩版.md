# AI模型配置快速参考 - 答辩版

## 🚀 5分钟快速配置指南

### 第1步: 获取API密钥

#### 通义千问 (推荐 ⭐)
1. 访问: https://dashscope.console.aliyun.com/apiKey
2. 登录阿里云账号
3. 创建API密钥
4. 复制 `sk-` 开头的密钥

#### 豆包 (可选)
1. 访问: https://console.volcengine.com/ark
2. 登录字节跳动账号
3. 创建推理接入点
4. 复制endpoint_id和API Key

### 第2步: 配置环境变量

```bash
# 复制环境变量模板
cp .env.ai.example .env

# 编辑.env文件,填入你的API密钥
DASHSCOPE_API_KEY=sk-你的密钥
```

### 第3步: 启动后端服务

```bash
cd backend
go run cmd/api/main.go
```

### 第4步: 测试AI功能

访问前端AI配置页面: http://localhost:3000/ai-config

## ⚠️ 常见问题快速解决

### Q1: API密钥无效
```
解决方案:
1. 检查密钥是否正确复制(不要有多余空格)
2. 确认密钥未过期
3. 检查账户余额或免费额度
```

### Q2: 连接超时
```
解决方案:
1. 检查网络连接
2. 增加超时时间配置
3. 使用代理服务器(如需要)
```

### Q3: 模型返回错误
```
解决方案:
1. 检查提示词是否符合规范
2. 确认模型名称正确
3. 查看后端日志获取详细错误信息
```

### Q4: 图像生成失败
```
解决方案:
1. 确认使用的是图像生成模型(qwen-image-plus)
2. 不要用视觉理解模型(qwen3-vl-plus)
3. 检查提示词长度和内容
4. 等待足够的生成时间(30-60秒)
```
