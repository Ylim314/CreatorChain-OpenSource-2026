# 开源代码与组件使用情况说明

本文档用于说明 CreatorChain 开源仓库中使用的主要开源组件、许可证情况与合规使用原则。

## 主要开源组件

- 后端：Gin、GORM、go-ethereum、go-redis
- 前端：React、Material UI、Ethers.js、Axios
- 合约：OpenZeppelin Contracts、Hardhat 及其插件

## 仓库许可证

- 本仓库采用 MIT 许可证，详见根目录 [LICENSE](../LICENSE)。

## 第三方组件许可证说明

- 各第三方依赖的具体许可证以其官方仓库/发布包声明为准。
- 使用者在二次分发或商用时，应自行审查并满足对应依赖的许可证要求。

## 使用边界

- 仓库中的示例配置、脚本和演示数据仅用于开发与学习。
- 部署到测试网/主网前，请替换密钥、RPC 和账户配置，不要复用演示参数。

## 安全与合规提醒

- 严禁将 `.env`、私钥、API 密钥提交到公开仓库。
- 如发现凭据泄露，应立即轮换并追溯相关访问日志。
