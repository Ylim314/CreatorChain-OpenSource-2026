## 🚀 快速开始指南

### 步骤1: 配置API密钥

```bash
# 1. 复制环境变量模板
cp .env.ai.example .env

# 2. 编辑.env文件
# 填入通义千问API密钥 (必须)
DASHSCOPE_API_KEY=sk-你的密钥

# 填入豆包API密钥 (可选)
ARK_API_KEY=你的密钥
```

**获取API密钥:**
- 通义千问: https://dashscope.console.aliyun.com/apiKey
- 豆包: https://console.volcengine.com/ark

### 步骤2: 启动服务

```bash
# 启动后端
cd backend
go run cmd/api/main.go

# 启动前端 (新终端)
cd client
npm start
```

### 步骤3: 测试AI功能

**方法1: 使用Web界面**
1. 访问 http://localhost:3000/ai-config
2. 选择模型 (推荐: qwen-plus)
3. 输入提示词测试

**方法2: 使用测试脚本**
```bash
cd backend/scripts
python test_ai_models.py
```

**方法3: 使用curl命令**
```bash
# 测试文本生成
curl -X POST http://localhost:8080/api/ai/generate \
  -H "Content-Type: application/json" \
  -d '{
    "prompt": "介绍一下区块链技术",
    "model": "qwen-plus",
    "task": "text"
  }'

# 测试图像生成
curl -X POST http://localhost:8080/api/ai/generate \
  -H "Content-Type: application/json" \
  -d '{
    "prompt": "一只可爱的卡通猫咪,赛博朋克风格",
    "model": "qwen-image-plus",
    "task": "image"
  }'
```

## ⚠️ 注意事项

### 1. API密钥安全
- ❌ 不要将API密钥提交到Git
- ✅ 使用.env文件管理密钥
- ✅ .env已在.gitignore中

### 2. 免费额度
- 通义千问: 新用户90天内100万tokens免费
- 图像生成: 100张免费(通义千问/万相)
- 建议: 答辩前先测试,确认额度充足

### 3. 网络连接
- 确保服务器能访问阿里云API
- 图像生成需要较长时间(30-60秒)
- 准备网络不稳定的备用方案

### 4. 错误处理
- 检查后端日志: `backend/logs/api.log`
- 查看浏览器控制台的Network请求
- 参考文档中的常见问题解决方案
