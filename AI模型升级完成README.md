## 🚀 立即开始使用

### 快速配置(3步)

```bash
# 1. 配置API密钥
cp .env.ai.example .env
# 编辑.env文件,填入你的通义千问API密钥

# 2. 启动后端
cd backend && go run cmd/api/main.go

# 3. 启动前端
cd client && npm start
```

### 获取API密钥

**通义千问** (必须,用于答辩)
- 访问: https://dashscope.console.aliyun.com/apiKey
- 免费额度: 100万tokens (90天有效)

**豆包** (可选)
- 访问: https://console.volcengine.com/ark
- 需要创建推理接入点
